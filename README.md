## Deskripsi
Repositori ini berisi catatan dan praktik dari mata kuliah **Analisis Eksplorasi Data (STA1232)** yang sedang saya tempuh. Mata kuliah ini berfokus pada pemahaman struktur data melalui statistik deskriptif, visualisasi, dan metode robust sebelum masuk ke tahap pemodelan.

## Materi Utama
- **Statistika Deskriptif:** Mean, median, kuartil, standar deviasi, skewness, dan kurtosis  
- **Visualisasi Data:** Histogram, boxplot, dan Kernel Density Estimation (KDE)  
- **Uji Normalitas:** QQ-Plot, Kolmogorov–Smirnov, Shapiro–Wilk, Anderson–Darling  
- **Outlier Detection:** Z-Score, IQR, MAD, dan Schwertman’s Fence  
- **Statistika Robust:** Trimmed mean dan winsorized mean  

## Tools

- NumPy, Pandas, Scipy, Matplotlib, Seaborn, ggplot2
